# Images Directory Structure

This folder contains all the images used throughout your portfolio.

## Folders:

### `/gallery`
- General portfolio images
- Project screenshots
- Work samples
- Any images you want to showcase

**Usage in code:**
```tsx
<img src="/images/gallery/your-image.jpg" alt="Description" />
```

### `/projects`
- Project-specific images
- Hero images for project pages
- Project thumbnails
- Technical diagrams

**Usage in code:**
```tsx
<img src="/images/projects/project-name.jpg" alt="Project" />
```

### `/certificates`
- Certificate images (PNG, JPG, PDF screenshots)
- Achievement badges
- Award images

**Usage in code:**
```tsx
<img src="/images/certificates/cert-name.jpg" alt="Certificate" />
```

## Recommended Image Formats:
- **JPG/JPEG**: Photos, screenshots (smaller file size)
- **PNG**: Graphics with transparency, logos, certificates
- **WebP**: Modern format (best compression)

## Naming Convention:
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive
- Examples:
  - `aws-cloud-practitioner.jpg`
  - `roblox-game-screenshot.png`
  - `project-dashboard-view.jpg`
