## Image Guidelines

- **Format**: JPG or PNG
- **Naming**: Use lowercase with hyphens (e.g., `construction-main.jpg`)
- **Size**: Optimize for web (recommended < 2MB per image)
- **Aspect Ratio**: 16:9 works best for project screenshots
- **Content**: Show key features, interfaces, and functionality
