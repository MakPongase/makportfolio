# EcoTide Adventures - Image Gallery

## Folder Structure

```
ecotide/
├── gameplay/          # Gameplay screenshots for Easy, Medium, Hard modes
├── setting/           # Philippine coastal setting visuals
├── education/         # SDG alignment and educational content images
└── ecotide-main-display.png  # Main featured image for the project page
```

## Image Naming Convention

- **Main Display**: `ecotide-main-display.png`
- **Gameplay**: `gameplay-easy.png`, `gameplay-medium.png`, `gameplay-hard.png`
- **Setting**: `setting-background.png`, `setting-elements.png`
- **Education**: `sdg-alignment.png`, `marine-conservation.png`

## Usage

All images should follow the lowercase-with-hyphens naming convention as per the portfolio standards.
