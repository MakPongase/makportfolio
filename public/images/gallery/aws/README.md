## Image Guidelines

- **Format**: JPG or PNG
- **Naming**: Use lowercase with hyphens (e.g., `amazon-q-1.jpg`)
- **Size**: Optimize for web (recommended < 2MB per image)
- **Aspect Ratio**: 16:9 or 4:3 works best
