# Image File Naming Guide

## Naming Convention:
- **Lowercase only**
- **Use hyphens** to separate words (not spaces or underscores)
- **Be descriptive** but concise
- **Format**: `project-name-description.ext`

## Common Suffixes:
- `-main-display` - Main/hero image
- `-screenshot` - General screenshot
- `-interface` - UI/Interface view
- `-dashboard` - Dashboard view
- `-thumbnail` - Smaller preview image
- `-hero` - Large banner image
- `-feature` - Feature showcase

## File Format Tips:
- `.jpg` - Photos, screenshots (smaller file size)
- `.png` - Graphics with transparency, UI screenshots
- `.webp` - Modern format (best compression)

## ✅ Good Examples:
- `roblox-main-display.jpg`
- `google-project-management.png`
- `calorie-tracker-dashboard.jpg`

## ❌ Avoid:
- `Roblox Game Main.jpg` (spaces, capitals)
- `roblox_game.jpg` (underscores)
- `IMG_1234.jpg` (not descriptive)
- `MyProject.PNG` (capitals)
